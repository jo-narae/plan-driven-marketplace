# plan-driven-marketplace

Claude Code 플러그인 마켓플레이스. `plan-driven-app-development` 스킬을 배포합니다.

## 설치

```
/plugin marketplace add <github-username>/plan-driven-marketplace
/plugin install plan-driven-app-development@plan-driven-marketplace
```

로컬에서 테스트하려면:

```
/plugin marketplace add /Users/ellen/test
/plugin install plan-driven-app-development@plan-driven-marketplace
```

## 수록 플러그인

| 플러그인 | 설명 |
|---|---|
| plan-driven-app-development | 새 앱을 7단계 기획 후 마일스톤으로 구현하도록 안내하는 스킬 |
